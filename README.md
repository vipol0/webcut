# WEBCUT

**WEBCUT** — простой видеоредактор, который работает целиком в браузере: обрезка по времени, кадрирование, громкость и экспорт в видео / MP3 / GIF. Файлы не покидают устройство пользователя, сервер не нужен.

**WEBCUT** is a simple video editor that runs entirely in the browser: trim by time, crop, adjust volume, and export to video / MP3 / GIF. Files never leave the user's device — no server required.
