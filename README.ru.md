# WEBCUT

Видеоредактор, который работает целиком в браузере — без установки, без загрузки файлов на сервер. Все операции (обрезка, кадрирование, изменение громкости, рендер) выполняются прямо на устройстве пользователя.

🌐 [English version](README.en.md)

## Возможности

- **Импорт** — перетащите файл или выберите его через диалог. Поддерживаются любые форматы, которые умеет проигрывать браузер (MP4, WebM, MOV, OGV и т.д.).
- **Обрезка по времени (trim)** — выбор начала и конца ролика прямо на шкале, с точным таймкодом.
- **Кадрирование** — рамка кадрирования прямо поверх видео, с изменением размера и перемещением мышью/пальцем.
- **Громкость** — регулировка уровня звука с мгновенным прослушиванием и «запеканием» в готовый файл.
- **Экспорт**:
  - **Видео** — MP4, MOV или WebM (набор доступных форматов зависит от браузера), с настройкой битрейта.
  - **Аудио** — MP3 с выбором битрейта (128 / 192 / 320 kbps).
  - **GIF** — анимация с выбором частоты кадров (5–15 fps).
- **Разрешение** — готовые пресеты 1080p / 720p / 480p / 360p (пересчитываются от текущей рамки кадрирования) или произвольное разрешение по ширине и высоте.
- **Скорость видео** — ускорение готового ролика в 1×–4×.
- **Сравнение размеров** — после экспорта показывается исходный и новый размер файла и разница между ними.
- **Локализация** — интерфейс на русском и английском языке, переключается кнопкой в шапке, выбор запоминается.

## Как это работает

Файл открывается локально прямо в браузере и никуда не отправляется. Кадры видео перерисовываются на `<canvas>` с учётом кадрирования и выбранного разрешения, а результат записывается через встроенные браузерные API (`MediaRecorder`, `Web Audio API`) — то есть браузер сам выступает кодировщиком.

Из ограничений: набор поддерживаемых на выходе форматов и скорость обработки зависят от конкретного браузера — лучше всего сайт работает в Chrome и Edge.

## Технологии

- Ванильный HTML / CSS / JavaScript, без сборки и зависимостей на этапе разработки.
- [`lame.min.js`](libs/lame.min.js) — кодирование в MP3.
- [`gifenc`](libs/gifenc.esm.js) — сборка GIF-анимации.

## Запуск

Проект не требует сборки — достаточно открыть `index.html` в браузере (или раздать папку любым статическим веб-сервером, если нужен доступ по HTTP).

```
index.html
libs/
  lame.min.js
  gifenc.esm.js
```

Важно, чтобы файлы `libs/lame.min.js` и `libs/gifenc.esm.js` лежали рядом с `index.html` в подпапке `libs` — иначе экспорт в MP3 и GIF не заработает.
